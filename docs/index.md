## Project Summary

**Cocktail Recipe Library** is a full-stack web application that:

1. Extracts cocktail recipes from screenshots using AI (Claude Vision)
2. Organizes recipes by category (template, spirit, glassware, etc.)
3. Provides a searchable, filterable library
4. Works offline with PWA support

### Tech Stack

| Layer | Technology |
|-------|------------|
| **Backend** | Python 3.9+, FastAPI, SQLAlchemy 2.0, PostgreSQL |
| **Frontend** | Next.js 14, React 18, TypeScript, Tailwind CSS |
| **AI** | Anthropic Claude Vision API (Haiku 4.5) |
| **Deployment** | Railway (backend + frontend + PostgreSQL) |

## Getting Started

### Prerequisites
- Python 3.9+
- Node.js 18+
- Anthropic API key

### Quick Start

```bash
# Backend
cd cocktail-app/backend
python3 -m venv venv
source venv/bin/activate
pip install -r requirements.txt
echo "ANTHROPIC_API_KEY=sk-ant-..." > .env
uvicorn app.main:app --reload

# Frontend (new terminal)
cd cocktail-app/frontend
npm install
npm run dev
```

- Backend: http://localhost:8000 (docs at /docs)
- Frontend: http://localhost:3000

## Core Flows

### 1. Recipe Extraction Flow

```
User uploads image
     │
     ▼
POST /api/upload/extract-immediate
     │
     ▼
Save image to filesystem
     │
     ▼
RecipeExtractor → Claude Vision API
     │
     ▼
Parse JSON response
     │
     ▼
Create Recipe + Ingredients in DB
     │
     ▼
Return RecipeResponse to frontend
```

### 2. Authentication Flow

```
User submits login form
     │
     ▼
POST /api/auth/login
     │
     ▼
Verify password hash
     │
     ▼
Generate JWT token
     │
     ▼
Frontend stores token
     │
     ▼
Include in Authorization header
```
